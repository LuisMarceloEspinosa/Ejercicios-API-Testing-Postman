Vamos a practicar lo aprendido en esta sección (Variables).


  1. En tu workspace, crear una nueva colección llamada "API cocteles"

  2. Dentro de esta actividad deberás crear 3 requests con las URL que estan acontinuacion y con las especificaciones que se detallan.

    1. Nombre de la request : Cocteles por nombre (variable colección)

      1. GET URL: www.thecocktaildb.com/api/json/v1/1/search.php?s=xxxx

      2. Se solicita crear una variable de coleccion con el nombre coctel y con un valor llamado mojito y pasarlo en la url del request en el parametro definido como "s"

      3. Si has hecho todo bien, cuando envies este request recibirás una respuesta 200 con la información del mojito en Postman.


  1. Nombre de la request : Buscar ingredientes por nombre (variables ambiente)

    1. GET URL: {{url}}/api/json/v1/1/search.php?i=yyy

    2. Dentro de tu workspace, crear un nuevo ambiente llamado "Ambiente apis cocteles" y

      1. Crear una variable llamada url y asignarle el valor actual www.thecocktaildb.com

      2. Crear una variable llamada "ingrediente" y asignarle el current value "vodka"

      3. Guarda el ambiente

      4. Para finalizar , debes referenciar la variable ingrediente en el path

        1. {{url}}/api/json/v1/1/search.php?i={{ingrediente}}

      5. Clic en guardar y enviar.

      6. Si has hecho todo bien, recibirás una respuesta 200 e información del ingrediente vodka en la respuesta.


    1. Nombre de la request : Buscar ingredientes por letra(variables globales)

      1. GET URL: www.thecocktaildb.com/api/json/v1/1/search.php?f=xxxx

      2. Dentro de tu workspace, crear una variable global llamada "letra-coctel" y asignarle un valor actual "a" y finaliza con guardar.

      3. Ahora debes referenciar la variable global en tu path

        1. www.thecocktaildb.com/api/json/v1/1/search.php?f={{letra-coctel}}


        2. Si has hecho todo bien, cuando envies este request recibirás una respuesta 200 con la información del mojito en Postman.
