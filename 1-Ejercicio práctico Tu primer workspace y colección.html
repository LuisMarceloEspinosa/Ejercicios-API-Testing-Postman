Vamos a practicar lo aprendido en esta sección.


  1. Crear una cuenta gratuita de Postman

  2. Descargar e instalar postman según la clase #8

  3. Crear un nuevo workspace llamado "Curso de Postman GeekQA"

  4. Crear una coleccion dentro de mi workspace llamado "colección API Libros"

  5. Dentro de la coleccion, crear una request con el nombre "Traer Libros" request tipo  GET a la API que esta en la siguiente dirección:"

    https://bookstore.demoqa.com/BookStore/v1/Books

  6. Dentro de la misma colección crear una carpeta llamada "Consulta libros" y alli se debe crear otra API request con el nombre Consultar libro x ISBN", request tipo GET a esta otra API:
    https://bookstore.demoqa.com/BookStore/v1/Book?ISBN=9781449325862


  7. Guardar todo y exportar la coleccion a una carpeta local dentro del comutador con el nombre "Coleccion API libros"

  8. Vuelve a tu workspace e importa de nuevo tu coleccion exportada y cambia el nombre a Colección API Libros favoritos"

    Realizar el ejercicio  y al finalizar puedes compararlo con la solución que tengo como archivo adjunto a esta clase. Solo debes importarla dentro de tu workspace tal como lo hemos aprendido en las clases anteriores
