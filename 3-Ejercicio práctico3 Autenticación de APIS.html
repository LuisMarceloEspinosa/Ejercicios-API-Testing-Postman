Vamos a practicar lo aprendido en esta sección (Autenticación de APIS).


  1. En tu workspace, crear una nueva colección llamada "Practica Autenticacion APIs Postman"

  2. Dentro de tu colección, crear  las siguientes requests:

    1. Nombre de la request : Autenticación basica

    2. GET URL: https://httpbin.org/basic-auth/

    3. En la pestana de autorización, seleccionar "Basic auth" y digitar en username "user", y password "passwd"

    4. Guardar la request y clic en send

    5. Validar que se reciba respuesta 200 - authenticated": true,


  3. Nombre de la request : Autenticación con token

    1. GET URL: https://httpbin.org/bearer

    2. En la pestana de autorización, seleccionar "Bearer token" y  pasar el siguiente token "foo"

    3. Guardar la request y clic en send

    4. Validar que se reciba respuesta 200 - respuesta authenticated": true,

  1. Nombre de la request : Openweather - Clima x ciudad con key value
